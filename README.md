## Get Started
### Install with npm

Run the following command in your terminal
```
npm install rww
```

### Import a JSON file

```JavaScript
var json = require('rww')('./test.json')
```

### Change the JSON

```JavaScript
json.random = Math.random()
```

The changes will now be reflected in the underlying file
